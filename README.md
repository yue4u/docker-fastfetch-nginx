# docker-fastfetch-nginx

Serve [fastfetch](https://github.com/fastfetch-cli/fastfetch) info via [aha](https://github.com/theZiz/aha) and [openresty](https://github.com/openresty).

[Docker image](https://hub.docker.com/r/yue4u/fastfetch-nginx)


```sh
docker run -p 3000:80 yue4u/fastfetch-nginx
```
